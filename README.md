# Supply-Chain-Risk-Assessment-Model
This project presents a data-driven Supply Chain Risk Assessment Model designed to identify, evaluate, and mitigate operational risks within a focused supply chain rather than a broad industry. The model analyzes critical disruptions expected in 2026, including raw material shortages, inflation-driven cost fluctuations, natural disasters, cybersecurity threats, transportation delays, supplier failures, and evolving global regulations. Microsoft Excel is used for data cleaning, preprocessing, and developing risk assessment matrices, while Python (Pandas) performs data modeling, failure-point analysis, and predictive risk scoring. Tableau provides an interactive dashboard that visualizes supplier risk, logistics bottlenecks, regional vulnerabilities, and high-risk components across the network. The project also incorporates a resilience framework featuring alternative supplier ("Plan B") strategies, vendor diversification, and mitigation planning to improve supply chain continuity. A weighted risk scorecard ranks potential disruptions based on probability and business impact, enabling organizations to prioritize corrective actions and optimize logistics decisions. By combining analytical thinking, predictive modeling, business intelligence, and real-world supply chain optimization techniques, this project demonstrates how modern data analytics can improve operational resilience, reduce costs, strengthen supplier management, and support informed decision-making. The solution reflects the practical skills employers seek in 2026, including data analysis, visualization, risk management, process optimization, and business problem-solving using industry-standard tools.
