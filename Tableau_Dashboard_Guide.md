# Tableau Dashboard Guide: Supply Chain Risk Assessment

This guide outlines the steps to build an interactive, professional **Supply Chain Risk Assessment Dashboard** in Tableau using the simulation outputs generated by the Python model.

---

## 1. Exported Datasets Overview
The simulation model exports four structured CSV files into the `data/` directory:
1. `tableau_summary.csv`: Aggregated scenario metrics (means, percentiles, stockout rates).
2. `tableau_simulation_runs.csv`: 1,000 randomized Monte Carlo iterations showing full cost/lead-time probability distributions.
3. `tableau_nodes.csv`: Geographic nodes (mining, refining, assembly) with coordinates, capacities, and calculated vulnerability indices.
4. `tableau_routes.csv`: Network arcs connecting the nodes, with baseline transit details.

---

## 2. Connect Data in Tableau

### Step 1: Connect to the Simulation Runs (Distribution Sheet)
1. Open Tableau and click **New Data Source**.
2. Select **Text File** and choose `data/tableau_simulation_runs.csv`.
3. Rename the connection to `Simulation Runs`.

### Step 2: Connect to Nodes & Routes (Network Map Sheet)
1. In the same Tableau workbook, add a new data connection.
2. Select **Text File** and connect `data/tableau_nodes.csv`.
3. In the Data Source canvas, drag in `data/tableau_routes.csv` and set up a relation or join:
   - Match `tableau_nodes.csv [Node_ID]` with `tableau_routes.csv [Origin]` (or `Destination` depending on map layering).
   - Alternatively, keep them as independent sheets and use **Blended Data** or Tableau's **Relationship model** linking them by `Node_ID`/`Origin`.

---

## 3. Sheet 1: Logistics Network Risk Concentration Map
*This sheet visualizes where bottlenecks and geopolitical vulnerabilities are concentrated.*

1. **Set Coordinates**: 
   - Drag `Longitude` to the **Columns** shelf.
   - Drag `Latitude` to the **Rows** shelf.
   - Change both coordinates to **Dimension** (right-click -> Dimension) instead of Measure (Average).
2. **Configure Node Visuals**:
   - Drag `Name` to the **Detail** card.
   - Drag `Vulnerability_Index` to the **Color** card. Use a custom Diverging palette: **Red-Green Gold** or a custom **Red-Orange-Blue** palette (with Red indicating high FEOC vulnerability).
   - Drag `Base_Capacity_Tons` to the **Size** card to represent production footprint.
3. **Overlay Routes (Dual Axis Map)**:
   - Hold `Ctrl` and drag another copy of `Latitude` next to the existing one on the Rows shelf.
   - On the second marks card, change the mark type to **Line**.
   - Drag `Route_ID` to **Detail**, and `Scenario` to **Color** (Red for As-Is, Green for To-Be).
   - Right-click the second `Latitude` pill on the Rows shelf and select **Dual Axis**.
4. **Format Map**: Clean up background maps (set to Dark Mode or Light Minimalist) and adjust line opacity to `50%`.

---

## 4. Sheet 2: Risk Scorecard Metric Comparison
*An interactive card table comparing key performance and risk indicators.*

1. Drag `Scenario` to the **Columns** shelf.
2. Drag `Measure Names` to the **Rows** shelf.
3. Drag `Measure Values` to the **Text** card.
4. Filter `Measure Names` to only include:
   - `Mean Total Cost TCOR` (Total Cost of Risk)
   - `Cost 90th Percentile VaR` (Value at Risk)
   - `Mean Total Lead Time`
   - `Lead Time 90th Percentile`
   - `Stockout Probability`
5. Right-click and format the values:
   - Costs as Currency (e.g., `$123,456` with 0 decimal places).
   - Lead times as Numbers (1 decimal place).
   - Stockout probability as Percentage (`0.0%`).

---

## 5. Sheet 3: Cost Risk Distributions (Monte Carlo Visual)
*Visualizes the tail-risk and cost volatility between the two strategies.*

1. Select the `Simulation Runs` data source.
2. Drag `Total_Cost_TCOR` to the **Columns** shelf. Right-click and select **Dimension** (or create a Bin of size $1,000 and place the bin on Columns).
3. Drag `Trial` (Count) or `Number of Records` to the **Rows** shelf. Change the measure to **Percent of Total**.
4. Drag `Scenario` to the **Color** card. Set:
   - `As-Is (Unmitigated)`: Soft Crimson (`#d9534f`)
   - `To-Be (Resilient)`: Soft Green (`#5cb85c`)
5. Add reference lines:
   - Right-click the X-axis -> **Add Reference Line**.
   - Set to **Average** of `Total_Cost_TCOR` per cell, sub-divided by Scenario.
   - Repeat to add a **90th Percentile** reference line to mark the Value-at-Risk (VaR) boundary.

---

## 6. Assemble the Dashboard
1. Create a new Dashboard (size: `1366x768` or generic Desktop).
2. Drag **Logistics Network Map** to the main pane.
3. Drag the **Cost Risk Distribution** and **Risk Scorecard** to the side/bottom panes.
4. Add a global filter on `Scenario` or setup interactive dashboard actions so clicking a node on the map filters the cost distribution sheet.
5. Apply a cohesive dark or clean professional font theme (such as *Segoe UI* or *Outfit*).
